Cables HDMI

HDMI (interfaz de medios de alta definición) es un tipo de cable que se utiliza para transmitir señales de audio y vídeo de alta definición, la señal de audio y video se puede transmitir fácilmente sin comprometer la calidad de las imágenes y se utilizan para conectar TV´s, reproductores de DVD, transmisores multimedia y otros dispositivos electrónicos.

Cable Ethernet

El cable Ethernet es un tipo de cable de red informática que se utiliza para una red cableada. El cable conecta los conmutadores, monitores, PC a la LAN (red de área local). La longitud y la durabilidad del cable Ethernet describen la calidad de la conexión.

Cable VGA

El cable VGA (Video Graphics Ray) es otro tipo de cable que se usa para enviar señales de video y para conectar el monitor y la CPU de un PC. El cable VGA también se puede utilizar en TV´s con HD .

Mouse

Un mouse es un dispositivo de entrada para computadoras que permite controlar el movimiento del cursor en la pantalla. Viene en diferentes tipos (óptico, láser), con botones para clic izquierdo/derecho y rueda de desplazamiento. Esencial para navegación web, trabajo de precisión y juegos, adaptándose a diversas necesidades y preferencias ergonómicas. .

Teclado

Un teclado es un dispositivo de entrada esencial para computadoras y dispositivos móviles, utilizado para introducir texto y comandos.

Laptop

Una laptop es una computadora portátil diseñada para ser transportada y utilizada en cualquier lugar. Combina pantalla, teclado, touchpad o mouse integrado en un único dispositivo.

Pulsera antiestatica

Una pulsera antiestática es un dispositivo usado por técnicos y ensambladores de equipos electrónicos para prevenir daños por descargas electrostáticas (ESD). Se conecta a tierra y disipa la electricidad estática del cuerpo humano.

Sopladores de cpu

Una sopladora para PC es una herramienta diseñada para limpiar eficazmente el polvo y los residuos de los componentes internos de una computadora. Utiliza aire comprimido controlado para eliminar la suciedad de ventiladores.

impresoras
